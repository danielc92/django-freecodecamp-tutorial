Summary

This web application is built using Django Framework in Python 3.7. It also uses Bulma CSS framework. This web application is based off and adapted from the tutorial Python Django Web Framework - Full Course for Beginners by FreeCodeCamp.

What is the purpose of this application?

This application purpose is to learn the Django framework from scratch. Previously I've worked in flask and so I'm expecting the transition to Django to be relatively less painful, having the base knowledge of web development and flask under my belt. It will also serve the purpose of comparing Flask and Django to see which framework is most suitable under a given scenario, working under the presumption that every framework has its strengths and weaknesses.
